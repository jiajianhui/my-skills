# 平面设计

这个 skill 专注于平面视觉输出，包括海报、主视觉、社媒图、宣传页、封面图、活动物料等。

它关注画面表达、版式、视觉冲击、信息主次和输出风格，不负责产品 UI 结构设计。

## 适用场景

- 海报设计
- 活动主视觉
- 社媒封面
- 宣传物料
- KV 方向探索
- 图文版式调整

## 工作流程

1. 明确物料类型和使用场景。
2. 明确主要信息和次要信息。
3. 判断需要突出的是品牌感、活动感、产品感，还是传播性。
4. 选择合适的画面策略：
   - 以排版为主
   - 以图像为主
   - 以图形为主
   - 以色块和构成为主
5. 给出清晰的版式和视觉方向建议。

## 输出要求

输出至少包含：

- 画面概念
- 版式结构
- 主视觉元素建议
- 色彩建议
- 字体建议
- 信息层级建议
- 应避免的问题

## 设计约束

- 信息主次必须一眼可见。
- 画面元素不要平均发力。
- 不要为了“丰富”而堆太多装饰。
- 如果传播目标明确，优先服务传播，而不是自我表达。
