# 品牌设计方向

这个 skill 专注于品牌视觉层面的方向定义，适用于新品牌建立、旧品牌升级、品牌语气统一、视觉系统方向整理等任务。

它关注的是“品牌给人的整体感觉和表达方式”，而不是具体网页 UI 布局，也不是单张海报的执行。

## 适用场景

- 品牌升级
- 新品牌视觉方向探索
- 品牌关键词提炼
- 品牌气质描述
- 视觉语言统一
- 品牌参考案例拆解

## 工作流程

1. 明确品牌对象：产品品牌、个人品牌、企业品牌、活动品牌，还是子品牌。
2. 明确品牌目标：建立信任、提高辨识度、强化高级感、增强科技感，还是走年轻化路线。
3. 提炼 3 到 5 个品牌关键词。
4. 用一句话定义品牌气质。
5. 给出品牌视觉系统建议：
   - 色彩倾向
   - 字体气质
   - 图形语言
   - 图片风格
   - 排版方式
6. 明确哪些表达应当避免，防止品牌跑偏。

## 输出要求

输出至少包含：

- 品牌气质一句话定义
- 品牌关键词
- 视觉方向描述
- 色彩与字体建议
- 图形与图片风格建议
- 应避免的风格倾向

## 设计约束

- 不要把“高级感”理解成单纯的黑金或奢华堆砌。
- 不要只给形容词，必须落到可执行的视觉判断。
- 品牌方向必须和产品定位、用户认知一致。
- 不要混用过多相互冲突的视觉人格。
